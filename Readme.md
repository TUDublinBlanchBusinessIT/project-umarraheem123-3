# **Gym Capacity Tracker**

This project is a simple PHP and MySQL web application. It allows users to record their daily meals and store them in a database. The system will display all saved meals.

## **Project Functionality**
- Records gym-goers entries and exits
- Stores all attendance data in a MySQL database
- Tracks current gym capacity in real time.
- Displays a log of all recorded entries and exits.
- Records the date and time of every entry and exit event.
